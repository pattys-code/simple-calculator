# Calculator

Very simple browser calculator. This project is part of the "Odin Project" curriculum.

*How to use:* Check out the latest GitHub pages deplyoment. You can enter numbers by clicking the buttons. Once you entered a number, an operator, another number and hit "=" the result is displayed. Entering a new number will start a new calculation. When you enter a number, an operator, another number and then again an operator, the current result is displayed but you can keep doing calculations by entering more numbers. Click around and have fun!

*Limitations:* It is a very simple browser calculator, so don't expect any advanced features. :) 
- you can only use integers
- the result will be rounded to three decimal places
- no backspace button
- no keyboard support
