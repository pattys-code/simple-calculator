# Calculator

Very simple browser calculator.This project is part of the "Odin Project" curriculum.

*How to use:* Check out the latest GitHub pages deplyoment. Click around and have fun!

*Limitations:* It is a very simple browser calculator, so don't expect any advanced features. :) 
- you can only use integers
- the result will be roudned to three decimal places
- no backspace button
- no keyboard support
